# IEEE-CIS-Fraud-Detection
Sistema de deteccion de fraude en transacciones de tarjetas de credito en linea desarrollado sobre el dataset IEEE-CIS Fraud Detection de Vesta Corporation. El sistema implementa un modelo de ensemble (LightGBM + XGBoost) con ingenieria de caracteristicas basada en la solucion ganadora de Chris Deotte (1er lugar, AUC=0.9459).
